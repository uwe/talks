\subsection{What is a segfault?}

\frame{
  \frametitle{What is a segfault?}

  \begin{itemize}
    \item segfault = segmentation fault
\pause
    \item every process has memory pages
    \item these pages are mapped to physical memory
\pause
    \item if you try to access an invalid address
    \item (or write to a protected address)
\pause
    \item BOOOM!
  \end{itemize}
}
